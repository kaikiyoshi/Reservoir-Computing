# Echo State Network (ESN)

This repository contains an implementation of Echo State Networks (ESNs) for solving tasks such as the NARMA benchmark.

## Overview
Echo State Networks are a type of reservoir computing model used for time-series prediction and sequence modeling. This project focuses on their application to the NARMA task, which is commonly used to evaluate the performance of machine learning models on nonlinear and memory-dependent problems.

## Features
- Implementation of ESN for the NARMA task
- Configurable hyperparameters
- Visualization of results

## File Structure
- `ESN/`: Contains the core implementation of the Echo State Network
  - `NRMA_task.py`: Script for training and evaluating the model on the NARMA task.
